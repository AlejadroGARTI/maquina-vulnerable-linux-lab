Ammonia-Lab es un entorno controlado de pruebas y aprendizaje en ciberseguridad diseñado bajo el formato de un reto CTF (Catch The Flag). El laboratorio simula un servidor empresarial basado en Ubuntu Server cuya finalidad principal es alojar un blog corporativo de divulgación científica en WordPress. La infraestructura expone públicamente dos puntos de entrada principales: un servicio de transferencia de hipertexto gestionado por el servidor web lighttpd a través del canal HTTP  en el puerto 7664, y un servicio de acceso remoto administrado por OpenSSH que opera sobre el canal SSH en el puerto 22.
